# javautil-jcmdline

Create command line arguments by creating an annotated commandline bean

## Getting Started

clone this repository and *mvn clean install*

## Example

### Arguments Bean

# Annotations
    @AcceptableValues
    @Argument
    @DependentField
    @DirectoryArguments
    @DirectoryExists
    @DirectoryReadable
    @DirectoryWritable
    @ExclusiveArguments
    @Exclusive
    @FieldValue
    @FileExists
    @FileReadable
    @FileWritable
    @Hidden
    @InheritenceArguments
    @IntegerArguments
    @MultiValueBean
    @MultiValue
    @Optional
    @OptionalRequiredArguments
    @package-info
    @Required
    @RequiredUnlessArguments
    @RequiredUnless
    @RequiresArguments
    @Requires
    @StringSet
