Reports

Reports are the answers to questions you should be asking.

Rather than provide a bunch of reports that the user needs to review to see if something requires action, Diamond APS provides reports of user specified conditions. These online reports link to the appropriate screens to review or take action.

Filters include

  • Vendor Code

  • Buyer allows a user to review the parts he is responsible for purchasing

  • Forecast Group Restrict to items applicable to a given forecast group, see Forecast Group
  • Manufacturer Code Restrict items to those available from a specified manufacturer
