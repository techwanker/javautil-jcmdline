Quantity Pricing
================

Let us assume that there is a setup cost and an incremental cost to
create a part such that the following relationship exists:

* u = unit cost 
* s = setup cost 
* i = incremental cost 
* n = number of units 
* t = total cost

This results in the very simple:

t = s + (n \* i)

The unit cost is then

u = t / n

or

u = (s + (n \* i)) / n
